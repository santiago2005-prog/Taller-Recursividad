Descripción del taller

Este taller tiene como objetivo aplicar el uso de funciones recursivas para resolver diferentes problemas de programación.

Se desarrollaron varios ejercicios donde se implementa la recursividad en casos como:

Cálculo de factorial
Inversión de números
Sumatorias
Operaciones matemáticas (multiplicación, división, potencia)
Manipulación de cadenas
Uso de arreglos y matrices
Serie de Fibonacci
Función de Ackermann

Cada ejercicio se encuentra en una clase independiente dentro del paquete Recursividad.

Instrucciones para ejecutar el programa

Clonar el repositorio:

Abrir el proyecto en un entorno de desarrollo como Visual Studio Code

Navegar a la carpeta:

src/Recursividad/
Ejecutar el archivo correspondiente al ejercicio que se desea probar:
Ejercicio1.java
Ejercicio2.java
...
Ejercicio14.java
Seguir las instrucciones que aparecen en consola (en los casos donde se solicita entrada de datos).
